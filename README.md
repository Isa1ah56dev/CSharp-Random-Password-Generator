# C# Password Generator

This is a very simple random password generator, its made in C#.
Its a console application

You can:
Choose which character types to include
Uppercase letters
Lowercase letters
Numbers
Special characters
Choose a custom password length

How it works is very simple, you launch the application from the exe file, and its pretty self explanatory.

# Requirements

You NEED .NET SDK / .NET Runtime

You can find the executable in the bin folder. This is open sourced so you can look into it, etc.
